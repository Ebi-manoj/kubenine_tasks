

## How request reaches container

```text
Browser → localhost:8080
        ↓
Host network stack
        ↓
Docker iptables (port mapping)
        ↓
docker bridge (docker0)
        ↓
Container IP:80
        ↓
NGINX worker process
        ↓
Response back to browser
```

### Flow explanation

- Browser sends request to port 8080  
- Docker maps it to container port 80  
- Request goes through docker bridge network  
- nginx inside container handles the request  
- Response is sent back to browser
